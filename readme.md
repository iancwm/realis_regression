# REALIS scikit-learn regression

This is a simple Jupyter notebook regression experiment on Singapore real estate transaction data.

## Setup

This project uses `uv` for dependency management. To install dependencies:

```bash
uv sync
```

To run the Jupyter notebook:

```bash
uv run jupyter notebook "REALIS Regression.ipynb"
```

## Requirements

- Python >= 3.12
- uv (https://github.com/astral-sh/uv)

## Dependencies

- numpy
- pandas
- scikit-learn
- matplotlib
- seaborn
- scipy
